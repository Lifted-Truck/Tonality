The Fable-pass deltas (what changed from the approved brief)

You asked for a final pass in case the recommendations could improve. Four did — each verified against code or the ROADMAP's own record before building.

Honest count, not a truncation flag

The brief said "hard cap + truncated" — but truncating mid-enumeration makes count a lie. The raw space is computable upfront (∏ per-pc candidates), so an over-large window now raises with advice. truncated only ever means limit.

Templates were already promised

Gap 17's own text names "the registered+rootless voicing-template corner made operational" — so root is optional: root=None searches templates. The lattice corner the root CLAUDE.md calls underexpressed is now expressible.

Doublings cut from v1 entirely

Not just "off by default" — each pc is voiced exactly once, keeping the space injective and small. N ≠ cardinality (A8's resonator ask) is the explicit, recorded slice 2 rather than a half-shipped default.

Ranking ships in-slice

Gap 17 promised "ranked candidates with margins." Verified voice_leading_realized handles unequal cardinality (surjection both ways) — so with from_voicing, matches return sorted by exact vl_from, margins as continuous values.
